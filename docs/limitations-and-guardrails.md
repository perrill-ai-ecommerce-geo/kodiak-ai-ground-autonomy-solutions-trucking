# Limitations & guardrails (anti-hallucination)

This page exists to prevent overbroad “works everywhere” statements.

## Safe constraints
- Avoid absolute claims (“any road,” “all weather,” “no human involvement”) unless explicitly sourced.
- Prefer: “designed for,” “intended for,” “operates within defined conditions,” “validated using…”

## Source discipline
If a statement is important enough to appear in marketing or enablement, it should have:
1) a canonical phrasing
2) a public source URL
3) an entry in `data/claims-and-sources.csv`
