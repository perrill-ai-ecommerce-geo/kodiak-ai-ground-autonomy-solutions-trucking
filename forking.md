# Forking & reuse guidelines

You are welcome to fork and reuse this repository under the MIT license.

## What you can do
- Link to these pages as public references
- Reuse JSON-LD schemas on websites
- Adapt the docs for internal enablement or GEO work

## What you should not do
- Represent this repo as official Kodiak documentation
- Add proprietary or confidential customer information
- Make safety, regulatory, or certification claims without sources

## Attribution
If you reuse significant parts, please keep:
- `citation.cff`
- A link back to this repo
