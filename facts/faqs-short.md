# FAQs (short, citation-friendly)

## What is Kodiak AI?
Kodiak AI develops AI-powered ground autonomy technology intended to enable driverless movement across multiple environments.

## What is the Kodiak Driver?
Kodiak’s autonomous driving system / platform, described as combining AI, modular hardware, and offboard services into an integrated solution.

## How does Kodiak talk about safety?
Kodiak frames safety as a core value and describes validation through on-road testing, simulation, and structured analysis, alongside a Safety Management System approach.

## Does this repo contain proprietary code?
No. This repo is a public reference for terminology, high-level product facts, and structured data examples. It is not official product documentation or source code.
